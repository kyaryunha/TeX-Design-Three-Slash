# TeX-Design-Three-Slash
Three Slash TeX Design ( Design by Me )



아래와 같은 디자인의 TeX 코드이다.

자세한 디자인은 ThreeSlashTeXDesign 에서 볼 수 있다.

스타를 누르던가 혹은 출처를 명시해주고 사용해주시면 감사합니다.
